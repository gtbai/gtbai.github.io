Safari中印象笔记·剪藏插件重新登录问题解决方法

May 17, 2017 | 日常技巧 | Hits

印象笔记（Evernote）是个好东西，其中有个最重要的原因就是它强大的浏览器剪藏插件（Evernote Web Clipper）。不过在macOS系统下Safari中的Evernote Web Clipper会有一个重新登录的问题。

具体来说，如果点击剪藏插件进行登录，那么下一次打开Safari的时候，插件会显示未登录。于是每一次打开浏览器后点击插件进行剪藏时都要重新登录，哪怕是有Safari的钥匙串，也会显得很麻烦。

具体的解决方法如下：

  1. 打开Evernote的官网，国际版用户点击这里，国内版印象笔记用户点击这里。
  2. 选中保存登录信息30天，登录账号。
  3. 这时你的剪藏插件就显示登录了，然后就算是关闭Safari重新打开，也会显示登录的状态。
